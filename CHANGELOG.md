# Changelog

All notable changes to the "Learning Git With Open Data" course are documented here.

Versioning: **major** for structural changes (adding or removing a module), **minor** for
new content within an existing module, **patch** for wording, link, or consistency fixes.

## [1.0.0] - 2026-07-21

### Added
- Initial release of the "Learning Git With Open Data" course: `agenda.md` plus six
  bilingual (EN/DE) module handouts (`handouts/00-setup.md` through
  `handouts/05-open-data.md`) covering setup, Git fundamentals, branching & merging,
  GitHub basics, Issues/PRs, and working with open data.
- Sample dataset (`data/bike-share-mini.csv`) used in the Module 5 exercise.
- `README.md` course overview and `LICENSE` (CC BY 4.0).
- Five ~90-minute sessions, run against the private practice repo
  `github.com/odon-at/education-github-playground`.
